# Security Policy

## Reporting Security Issues

The Copa maintainers take security seriously. We appreciate your efforts to responsibly disclose your findings.

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, please report them through our [private vulnerability reporting](https://github.com/project-copacetic/copacetic/security/advisories/new) form.

For more details, see the [Copacetic security policy](https://github.com/project-copacetic/copacetic/blob/main/SECURITY.md).
